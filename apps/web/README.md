# LLM Reliability Lab Web

This directory is reserved for the future Vite + React + TypeScript dashboard.

The frontend is intentionally not generated in Milestone 0.

Future dashboard areas:

- Scenario runner.
- Trace viewer.
- Evaluation dashboard.
- Demo agent interface.

The web app will call the Node.js API in `apps/api`. Backend behavior belongs in the API service, not in the frontend.
